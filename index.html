Natural Language Processing Application

This program is a simple speech-to-text application that allows a user to speak a sentence into their computer's microphone, and then displays the transcript of the spoken sentence on the screen, along with a list of the individual words in the sentence, with each word surrounded by inverted quotes.

The program uses the speech_recognition and nltk libraries to handle the speech recognition and tokenization, respectively. It also uses the tkinter library to create a graphical user interface (GUI) with a prompt, a button to start the speech recognition, and labels to display the transcript and tokenized words. When the user clicks the "Start Speaking" button, the program records their voice, transcribes it to text, tokenizes the text into individual words, and then displays the transcript and tokenized words in the GUI.
